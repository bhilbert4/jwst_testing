Usage
=====

The test suite can be run from within the ``jwst_testing/`` directory with a configuration as follows

.. code::
    
    $ py.test -v --config_file /path/to/my_config.cfg

The configuration file should have sections corresponding to the step name with input and ouptut products for each step.
For example

.. code::

    # Config file used to specify which tests will be run and which
    # files will be used for each of those tests.

    # [step_name]
    # input_file = path to input for that step
    # output_file = path to output for that step

    [dq_init]
    input_file = /path/to/your/data_uncal.fits
    output_file = /path/to/your/data_dq.fits

    [saturation]
    input_file = /path/to/your/data_dq.fits
    output_file = /path/to/your/data_sat.fits

    [refpix]
    input_file = /path/to/your/data_sat.fits
    output_file = /path/to/your/data_refpix.fits

    [reset]
    input_file = /path/to/your/data_refpix.fits
    output_file = /path/to/your/data_reset.fits

    [lastframe]
    input_file = /path/to/your/data_reset.fits
    output_file = /path/to/your/data_lastframe.fits

    [linearity]
    input_file = /path/to/your/data_lastframe.fits
    output_file = /path/to/your/data_lin.fits

    [dark_current]
    input_file = /path/to/your/data_lin.fits
    output_file = /path/to/your/data_dark_current.fits
